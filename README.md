# ML Practice Repository

**End-to-End Machine Learning Projects**  
Following *Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow* by Aurélien Géron.

## Overview
This repository contains my hands-on practice with core machine learning concepts, focusing on **regression** and **classification** techniques. The work demonstrates a complete ML workflow: problem framing, exploratory data analysis, data preprocessing, feature engineering, model training, hyperparameter tuning, and evaluation.

## Key Solo Project – Chapter 2: End-to-End Machine Learning
- **California Housing Dataset** (regression task)  
- Predicts median house values using census data  
- Full pipeline implemented (data loading → cleaning → modelling → evaluation)  
- Best model: Random Forest Regressor (saved as `my_carnifonia_housing_model.pkl` and `full_pipeline.pkl`)  
- Key skills practised: pandas, scikit-learn, matplotlib, pipeline construction, cross-validation, RMSE/MAE evaluation

**Relevance to Agricultural AI**  
The structured workflow developed here is directly transferable to real-world farming applications such as crop yield forecasting and personalised farming advice in low-resource environments — the exact type of work required at FARMA.
